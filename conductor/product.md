# Product Definition

## Initial Concept
A framework and playbook for building distributed AI agents using the Google Agent Development Kit (ADK) and Python. It is designed for Agent-Driven Development, providing the architectural guardrails, playbooks, and AI instructions needed to safely direct AI assistants to write code.

## Target Audience
- **Beginner AI Developers**: Developers who are comfortable with Git, CLI, and prompt engineering but have zero familiarity with developing AI agents.

## Core Capabilities
- **Rapid Prototyping**: Enabling developers to quickly scaffold and test new agent concepts.
- **Educational Platform**: Serving as a structured learning environment to master agent architecture and best practices.

## Agent Interaction Modes
The framework supports building agents across a wide spectrum of interaction models:
- **CLI Agents**: Command-line tools and utilities.
- **API/Web Agents**: Services exposing RESTful endpoints via FastAPI.
- **Autonomous Workers**: Background processes and asynchronous tasks.

## Key Differentiators
- **Architectural Guardrails**: Enforces strict Hexagonal Architecture to ensure clean separation of concerns and maintainability.
- **Playbooks & AI Prompts**: Provides comprehensive developer guides and system instructions to guide both humans and AI coding assistants.
- **Infrastructure Integrations**: Features out-of-the-box support for core infrastructure components like Redis and Postgres, abstracting complexity via the Universal Core.
