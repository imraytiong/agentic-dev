# Technology Stack

## Programming Language
- **Python 3**: The core language for building the framework and agents.

## Frameworks & Libraries
- **Google Agent Development Kit (ADK)**: The foundational framework for agent development.
- **FastAPI**: Used for building API and Web-based agent endpoints.
- **LiteLLM**: For standardized LLM API interactions.
- **Pydantic**: For robust data validation and schema definitions.

## Testing & Observability
- **Pytest**: The primary testing framework.
- **Arize-Phoenix**: For observability and tracing.

## Infrastructure (via Universal Core)
- **Redis**: For caching and fast ephemeral state.
- **Postgres**: For persistent storage.
